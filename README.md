天机综合排盘 隐私政策

最后更新日期：2025年7月1日

欢迎使用天机综合排盘（以下简称"本应用"）。本隐私政策旨在说明我们如何处理您的信息。

一、信息收集
本应用不收集任何个人信息。所有排盘计算均在您的设备本地完成，无需联网，我们无法访问您的任何数据。

二、本地存储
本应用仅在您的设备本地存储以下信息：
• 排盘历史记录
• 购买状态
• 语言偏好设置
以上数据仅存储在您的设备上，不会上传至任何服务器。

三、网络请求
本应用不发送任何网络请求，不与任何第三方服务器通信，不包含任何广告SDK或数据分析SDK。

四、第三方服务
本应用使用苹果App内购买（IAP）服务处理付款，该服务由Apple Inc.提供，受苹果隐私政策约束。

五、儿童隐私
本应用不收集任何用户信息，适合所有年龄段用户使用。

六、权限说明
本应用仅使用设备系统时钟用于排盘计算，不申请相机、麦克风、位置、通讯录等任何敏感权限。

七、政策变更
如本隐私政策发生变更，我们将在应用更新说明中告知用户。
